# Personal Portfolio Website

My personal portfolio website with a cyberpunk aesthetic showcase my projects, resume and technical skills. 

## Tech Stack

- **Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Animation:** Framer Motion
- **Icons:** React Icons

## Getting Started

### Prerequisites

- Node.js (v18 or later recommended)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
